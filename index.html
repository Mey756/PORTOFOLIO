Juli 2024

Instalasi Wordpress di AWS menggunakan 2 VPS (VPS 1 sebagai Webserver dan VPS 2 sebagai Database)

( Cloud Computing )

Ini project kedua kami, saya dan rekan tim saya menginstal WordPress di AWS menggunakan dua VPS. Saya mengelola VPS 2 sebagai database server, menginstal MariaDB di AWS RDS, mengonfigurasi storage, dan menginstal phpMyAdmin. Saya juga mengatur security group agar VPS 2 dapat berkomunikasi dengan VPS 1, serta mengubah konfigurasi bind-address di file 50-server.cnf agar database dapat diakses oleh semua sumber. Teman saya mengelola VPS 1 sebagai web server, menginstal Apache, PHP, WordPress, dan Webmin untuk pengelolaan server. Dia juga mengonfigurasi firewall pada VPS 1 dan melakukan pointing domain agar mengarah ke IP VPS 1. Setelah itu, saya mengonfigurasi koneksi antara VPS 1 dan VPS 2 di Webmin serta mengatur file wp-config.php agar WordPress dapat mengakses database dari VPS 2. Kedua VPS ini akhirnya terhubung dan berjalan lancar.
